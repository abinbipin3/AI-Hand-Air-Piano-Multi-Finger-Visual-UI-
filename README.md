# 🎹 AI Hand Air Piano (Multi-Finger + Visual UI)

An interactive AI-based Air Piano built using:
- OpenCV
- MediaPipe
- Pygame

## 🚀 Features
✔ Multi-Finger Detection  
✔ Real-time Hand Tracking  
✔ Visual Piano UI  
✔ Highlight Pressed Keys  
✔ Smooth Sound Playback  

## 🧠 Tech Stack
- Python
- OpenCV
- MediaPipe
- Pygame

## 🎥 How It Works
The system tracks finger landmarks using MediaPipe Hands and maps each finger to a piano key:

Thumb → C  
Index → D  
Middle → E  
Ring → F  
Pinky → G
